# 🚀 CareerOS AI

GenAI-powered Career Intelligence Engine that helps students make structured, data-driven career decisions.

---

## 🎯 Problem
Students often choose careers based on trends, peer pressure, or incomplete information, leading to skill mismatch and poor career outcomes.

---

## 💡 Solution
CareerOS AI provides:
- Structured career recommendations
- Personalized roadmap (6 months, 1 year, 3 years)
- Skill development guidance
- Alternative career pathways

---

## 🛠️ Features
- Dynamic career suggestions based on user input
- Structured roadmap generation
- Domain-specific career mapping
- Clean and interactive UI

---

## ⚙️ Tech Stack
- Python
- Streamlit

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
